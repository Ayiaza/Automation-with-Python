# 📂 Automated Daily Log Creator

A simple Python script that instantly creates a folder with pre-formatted daily text files (Monday to Friday) and lets you type custom tasks for each day.

## 🚀 How to Use

1. Save the code as `create_logs.py`.
2. Open your terminal and run:
   ```bash
   python create_logs.py
   ```
3. Enter a name for your new folder when prompted.
4. Type your main task for each weekday.

## 📁 Project Structure

```text
├── Your_Folder_Name/
│   ├── Monday.txt
│   ├── Tuesday.txt
│   ├── Wednesday.txt
│   ├── Thursday.txt
│   └── Friday.txt
└── create_logs.py
```

## 📄 File Example (`Monday.txt`)
```text
--- Notes for Monday ---

Tasks to complete:
- [Your custom task here]
```
